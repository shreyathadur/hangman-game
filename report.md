# HANGMAN GAME

## Abstract

The Hangman Game is a text-based word guessing game developed using Python. The player guesses a hidden word one letter at a time. The game allows six incorrect attempts before ending.

## Objectives

- Learn Python programming concepts
- Implement loops and conditions
- Use lists and strings
- Create an interactive console game

## Technologies Used

- Python 3
- Random Module

## Features

- Random word selection
- Input validation
- Six attempts
- Win/Loss condition

## Conclusion

The project successfully demonstrates Python fundamentals through an interactive game.
